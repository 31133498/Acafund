# AcaFund

The financial operating system for African student communities.

Collect, verify, govern, and report on every naira your class ever touches.

## Stack

- **Frontend** — React + TypeScript + Vite + Tailwind CSS + Lucide React
- **Backend** — Python / FastAPI + PostgreSQL + Docker

## Getting started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
cp .env.example .env   # fill in your secrets
docker compose up
```
