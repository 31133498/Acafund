# AcaFund

The financial operating system for African student communities. Collect,
verify, govern, and report on every naira your group ever touches.

## Stack

| Layer | Tech |
|-------|------|
| API | Python 3.11 · FastAPI · SQLAlchemy 2.0 |
| Database | PostgreSQL 16 (SQLite in-memory for tests) |
| Payments | Monnify (NGN collections) |
| AI | Anthropic claude-sonnet-4-6 (treasury assistant) |
| Container | Docker / docker-compose |
| Deploy | Render (Blueprint via render.yaml) |
| Frontend | React · TypeScript · Vite · Tailwind CSS |

## Local development

### Prerequisites
- Docker Desktop running
- Copy `.env.example` → `.env` and fill in secrets (see env-var table below)

```bash
cd backend
cp .env.example .env
# edit .env — at minimum set SECRET_KEY and (optionally) Monnify + Anthropic keys

docker compose up          # starts postgres + api on http://localhost:8080
```

API docs auto-generated at **http://localhost:8080/docs**.

### Running tests

Tests run against SQLite in-memory — no Postgres needed.

```bash
docker compose exec api python -m pytest tests/ -v
```

To skip only the live Monnify sandbox test:

```bash
docker compose exec api python -m pytest tests/ --ignore=tests/test_monnify_live.py -v
```

To run the live Monnify smoke test (requires real sandbox credentials in `.env`):

```bash
docker compose exec api sh -c "RUN_LIVE_MONNIFY_TESTS=1 python -m pytest tests/test_monnify_live.py -v -s"
```

## Environment variables

| Variable | Required | Description |
|---|---|---|
| `DATABASE_URL` | ✅ | PostgreSQL DSN, e.g. `postgresql://user:pass@host/db`. Auto-set by Render from the managed DB. |
| `SECRET_KEY` | ✅ | JWT signing secret — minimum 32 chars. Auto-generated by Render on first deploy. |
| `ACCESS_TOKEN_EXPIRE_MINUTES` | | JWT lifetime in minutes. Default: `60`. |
| `ALGORITHM` | | JWT algorithm. Default: `HS256`. |
| `MONNIFY_API_KEY` | ✅ (payments) | Monnify merchant API key. |
| `MONNIFY_SECRET_KEY` | ✅ (payments) | Monnify merchant secret — used for webhook HMAC verification. |
| `MONNIFY_CONTRACT_CODE` | ✅ (payments) | Monnify contract code for your merchant account. |
| `MONNIFY_BASE_URL` | | Monnify API base URL. Default: `https://sandbox.monnify.com`. **Change to `https://api.monnify.com` for production.** |
| `ANTHROPIC_API_KEY` | ✅ (assistant) | Anthropic API key for the treasury AI assistant. |
| `FRONTEND_ORIGIN` | | CORS allowed origin. Default: `http://localhost:3000`. Set to your deployed frontend URL. |
| `EXPENSE_APPROVAL_THRESHOLD` | | Informational threshold (NGN). Default: `50000`. |

## API overview

| Router | Prefix | Auth | Purpose |
|---|---|---|---|
| auth | `/auth` | — / Bearer | Register, login, `GET /auth/me` |
| communities | `/communities` | Bearer | Create, join, manage roles |
| collections | `/communities/{id}/collections` | Bearer | Create dues collections, close |
| payments | `/collections/{id}/pay` | Bearer | Initiate Monnify checkout |
| webhooks | `/webhooks/monnify` | None (HMAC) | Monnify payment event receiver |
| expenses | `/communities/{id}/expenses` | Bearer | Submit / approve / reject |
| reports | `/collections/{id}/transparency` | **None** | Public accountability report |
| reports | `/communities/{id}/dashboard` | Bearer | Treasury snapshot |
| assistant | `/communities/{id}/assistant/ask` | Bearer | AI treasury Q&A |

## Deploying to Render

### First deploy

1. Push this repo to GitHub.
2. In the Render dashboard click **New → Blueprint** and connect the repo.
   Render detects `render.yaml` automatically.
3. Render will prompt you for every `sync: false` env var — fill in:
   - `MONNIFY_API_KEY`
   - `MONNIFY_SECRET_KEY`
   - `MONNIFY_CONTRACT_CODE`
   - `ANTHROPIC_API_KEY`
   - `FRONTEND_ORIGIN` (your Vercel/Netlify URL or custom domain)
4. Click **Apply** — Render builds the Docker image and runs database migrations
   via SQLAlchemy `create_all` on first startup.

### Monnify webhook (after first deploy)

Once the service is live, copy the Render service URL and register the
webhook in the Monnify merchant dashboard:

```
https://<your-render-url>/webhooks/monnify
```

### Going live on Monnify production

1. In the Render dashboard update `MONNIFY_BASE_URL` from
   `https://sandbox.monnify.com` → `https://api.monnify.com`.
2. Swap `MONNIFY_API_KEY`, `MONNIFY_SECRET_KEY`, and `MONNIFY_CONTRACT_CODE`
   for your **production** merchant credentials.
3. Re-register the webhook URL in the Monnify **production** merchant dashboard.
